# Control-de-productos-
Practica de Power Bi
Control de Productos - Power BI Dashboard
Por: Juan Jesus Antonio Gonzalez Gonzalez

Este proyecto presenta una solución de Business Intelligence diseñada para el monitoreo y gestión eficiente de productos. El dashboard permite transformar datos transaccionales en indicadores clave de rendimiento (KPIs) para la toma de decisiones estratégicas.

 Objetivos del Proyecto
Visibilidad Total: Centralizar la información de stock, ventas y categorías de productos.

Optimización de Inventario: Identificar productos de baja rotación o quiebres de stock.

Análisis de Rentabilidad: Visualizar cuáles productos o familias generan el mayor margen de contribución.

 Herramientas y Tecnologías
Power BI Desktop: Construcción de visualizaciones y diseño de interfaz (UI).

Power Query (M): Limpieza, normalización y transformación de los datos de origen.

DAX (Data Analysis Expressions): Creación de medidas calculadas (Ventas Totales, YoY%, Promedios Móviles).

Modelado de Datos: Implementación de un esquema en estrella (Star Schema) para un rendimiento óptimo.

 Funcionalidades Clave
Filtros Dinámicos: Segmentación por fecha, categoría, proveedor y ubicación geográfica.

KPI Cards: Visualización rápida de métricas principales (Total Productos, Valor del Inventario, Margen Medio).

Análisis de Tendencias: Gráficos de líneas para observar el comportamiento de los productos a través del tiempo.

Matriz de Detalles: Tabla interactiva con formato condicional para identificar anomalías rápidamente.
